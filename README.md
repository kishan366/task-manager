# Task Manager

A beautiful, responsive, and modern Task Manager built with HTML, CSS, and vanilla JavaScript. 

This project was designed with a focus on visual excellence, featuring a sleek dark mode, vibrant gradients, glassmorphism, and smooth micro-animations. It allows you to quickly track, manage, and categorize your daily objectives.

## Features

- **Add Tasks**: Quickly create new tasks and assign them to specific categories (Work, Personal, Urgent).
- **Manage Tasks**: Mark tasks as completed or delete them when they are no longer needed.
- **Categorization**: Visual tags with icons make it easy to identify the context of each task.
- **Persistent Storage**: Uses your browser's `localStorage` to save your tasks so you don't lose them when you refresh the page.
- **Modern Aesthetics**:
  - Premium dark UI with indigo and purple gradient accents.
  - Glassmorphism effects for a sleek, layered look.
  - Smooth entrance animations and hover effects.
  - Clean and modern typography using Google's **Inter** font.

## Technologies Used

- **HTML5**: For the semantic structure of the app.
- **CSS3**: For advanced styling, flexbox/grid layouts, animations, and custom CSS variables.
- **JavaScript (ES6+)**: For DOM manipulation, event handling, and state management without any external frameworks.

## Getting Started

1. Clone or download this repository to your local machine.
2. Open the project folder.
3. Double-click the `index.html` file to open it in your default web browser. 

No build tools, package managers, or server setups are required!

## Project Structure

- `index.html`: The main entry point of the application containing the layout and structure.
- `style.css`: Contains all the styling, including variables, animations, and responsive design rules.
- `script.js`: Handles the application logic, event listeners, and local storage integration.
